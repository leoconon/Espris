# ESPris
#### Tetris no ESP32

## Como jogar
Montar o circuito com dois botões:
 - GPIO 2: Botão que gira a peça
 - GPIO 14: Botão que move a peça para a direita
 
Configurar a variável no **menuconfig**: `ESPris Configuration` > `ESPRIS_USERNAME`
